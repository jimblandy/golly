This directory contains the source files needed to embed Lua in Golly. It's essentially the same set of files from the src directory in the full distribution of Lua 5.3.4 (available here) but with these minor changes:

  • Removed lua.c (the Lua stand-alone interpreter).
  • Removed luac.c (the Lua compiler).
  • Simplified Makefile (so it can be called from makefile-mac or makefile-gtk).
